# nightowl


Run all the test cases.

###### ``` ./nightowl ```


Run specific test cases.

###### ``` ./nightowl tests/rack/test1.rb```
##### ``` ./nightowl tests/rack/test1.rb tests/test2.rb```


Skip specific test cases.

###### ``` ./nightowl  --skip test1``` 
###### ``` ./nightowl  --skip test1 test2```
